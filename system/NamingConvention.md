# Naming Convention

## General

Use English for all folders and system documents.

Use descriptive names.

Avoid spaces.

## Files

Use PascalCase for important documents.

Examples:

README.md

MasterContext.md

BrandGuide.md

Vision.md

Workflow.md

DecisionFramework.md

ProjectRoadmap.md

## Agent Files

Every agent should contain:

AGENT.md

workflow.md

inputs.md

outputs.md

examples.md

## Folders

Use lowercase.

Examples:

agents/

knowledge/

projects/

templates/

automation/

marketing/

music/

youtube/

## Project Names

Use kebab-case.

Example:

oskar-corbenn

future-project

youtube-growth

## Git

Commit messages should be concise.

Examples:

Create marketing strategist

Update brand guide

Improve workflow

Add research templates
