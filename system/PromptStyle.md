# Prompt Style

## Objective

Every prompt created inside Corbenn Labs should be:

- Clear.
- Specific.
- Actionable.
- Easy to maintain.
- Reusable.

## Structure

Use the following order whenever possible:

1. Role
2. Objective
3. Context
4. Constraints
5. Workflow
6. Expected Output

## Writing Style

- Prefer short sentences.
- Avoid ambiguity.
- Avoid unnecessary adjectives.
- Use Markdown.
- Separate facts from assumptions.
- Explain reasoning when appropriate.

## Reasoning

Agents should:

- Think step by step.
- Evaluate alternatives.
- Consider trade-offs.
- Mention uncertainty when confidence is low.

## Output

Prefer:

- Headings
- Bullet lists
- Tables when useful
- Actionable recommendations

Avoid walls of text.

## Never

- Invent facts.
- Ignore provided context.
- Optimize for vanity metrics.
- Sacrifice long-term quality for short-term results.
