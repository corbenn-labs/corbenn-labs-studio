# Production Rules

Version: 1.0

Purpose

This document defines the production rules learned from real-world editing sessions.

These rules are not theoretical guidelines. They represent validated production knowledge and must be treated as the source of truth for all future Short generation.

---

# Rule 01 — Verify the Audio First

Never assume the duration of a lyrical hook.

Always locate the hook inside the song and measure its actual duration before creating a storyboard.

Required Output

- Hook Start
- Hook End
- Total Duration

---

# Rule 02 — Audio Determines the Storyboard

The storyboard must adapt to the available audio.

Never extend the visual sequence simply to reach an arbitrary target duration.

If the emotional idea is complete in seven seconds, the Short should remain seven seconds.

---

# Rule 03 — Classify the Emotional Type

Before creating any storyboard, classify the Short.

Examples

- Impact
- Reflection
- Metaphor
- Identification
- Narrative
- Cinematic

The emotional category determines pacing, clip selection, and text rhythm.

---

# Rule 04 — Generate a Production Storyboard

Never return only creative ideas.

Always generate a production-ready storyboard.

Each storyboard must include:

- Clip Number
- Start Time
- End Time
- Suggested Footage
- On-screen Text
- Transition Notes (if needed)

---

# Rule 05 — Prioritize Existing Footage

Always use the project's existing media library before suggesting new assets.

The agent should recommend clips already available whenever possible.

Example

- Rain
- Forest
- Silhouettes
- House
- Broken Glass
- Roads
- Hands
- City
- Windows

---

# Rule 06 — Verify Text Readability

Text readability is mandatory.

Before recommending a shot, verify that the background provides sufficient contrast.

If readability may be compromised, suggest one of the following:

- Different clip
- Darkened background
- Repositioned text
- Subtle shadow behind text

---

# Rule 07 — Keep One Emotional Idea

Each Short should communicate one emotional message.

Avoid combining multiple lyrical ideas into a single edit.

---

# Rule 08 — Follow the Official Visual Standard

Unless explicitly overridden, every Short should follow the official production standard.

Timeline

- 1080 × 1920
- 30 FPS

Typography

- Open Sans SemiBold
- White
- Minimal animation
- One phrase per screen

Branding

- Oskar Corbenn logo
- Bottom-right corner
- 14% scale
- 75% opacity

---

# Rule 09 — Export Standard

Default export settings

- MP4
- H.264
- 1080 × 1920
- 30 FPS
- Constant Bitrate
- 15000 Kbps

Expected file size

10–30 MB

---

# Rule 10 — Simplicity Wins

Prioritize emotional clarity over visual complexity.

Avoid unnecessary effects, transitions, or distractions.

The song is the protagonist.

The visuals exist to reinforce the emotion.

---

# Rule 11 — Learn From Production

Every completed Short is a learning opportunity.

New discoveries must be documented and incorporated into these Production Rules.

This document evolves through real production experience, not assumptions.

---

# Core Principle

The goal of the Content Engine is not to generate ideas.

Its goal is to generate production-ready editing instructions that minimize creative friction while preserving the artistic identity of Oskar Corbenn.
