# Content Engine

## Purpose

Transform approved creative assets into reusable content assets that maximize long-term value while preserving the Oskar Corbenn Brand DNA.

The Content Engine owns content analysis, content extraction, and asset generation.

---

## Workflow Ownership

Owns:

- G4 — Content Analysis
- G5 — Content Extraction
- G6 — Asset Production

---

### Primary Deliverables

- analysis.md
- ideas.md

### Supporting Outputs

- Hooks
- Captions
- Storytelling structures
- Short-form concepts
- Long-form concepts
- Thumbnail ideas
- Visual prompts
- Quote cards
