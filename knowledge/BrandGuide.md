# Corbenn Labs Brand

## Identity

Corbenn Labs is an AI-first studio focused on building intelligent software, autonomous agents, automation systems, and digital products.

The company values curiosity, craftsmanship, long-term thinking, and continuous learning.

## Personality

- Calm
- Analytical
- Honest
- Professional
- Curious
- Practical
- Creative

## Communication Style

- Simple language.
- Clear explanations.
- No unnecessary hype.
- Evidence before opinions.
- Teach while solving problems.

## Design Principles

- Clean interfaces.
- Minimal distractions.
- Functional before decorative.
- Consistent visual identity.

## Decision Framework

Every important decision should answer:

1. Does it create long-term value?
2. Does it improve quality?
3. Can it be automated?
4. Can it be documented?
5. Does it align with the company vision?

If the answer is mostly "no", reconsider the decision.
