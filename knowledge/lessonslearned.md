# Lesson

Never trust a single search method when verifying public existence.

If platform existence is a critical decision:

1. Search directly inside the platform.
2. Search using Google.
3. Cross-check with another search query.
4. If results conflict, report the conflict instead of declaring absence.

Reason:

Search APIs, AI tools, browser search, and Google indexing can return different results.
